# 女神节祝福卡片生成器

一个简单而美观的网页应用，用于创建和分享女神节（3月8日国际妇女节）祝福卡片。

## 功能特点

- 个性化祝福卡片创建
- 多种精美主题风格选择
- 支持自定义头像上传
- 实时预览卡片效果
- 一键下载生成的卡片

## 使用方法

1. 在"致："栏中输入您想要祝福的人
2. 上传一张头像图片（可选）
3. 在祝福语文本框中输入您的祝福内容
4. 在"署名"栏中填写您的名字
5. 选择一个您喜欢的主题风格
6. 预览生成的卡片
7. 点击"下载卡片"按钮保存您的创作

## 主题风格

当前支持以下主题风格：
- 清新淡雅
- 活力阳光
- 浪漫甜蜜
- 典雅沉稳
- 晚霞余晖
- 薰衣草紫

## 技术栈

- HTML5
- CSS3
- JavaScript
- HTML2Canvas (用于将卡片导出为图片)
